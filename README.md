# 🎮 Tamagosol

> Raise your crypto pet, earn $SOGO, evolve to greatness.

**Live at:** [tamagosol.fun](https://tamagosol.fun)

## What is Tamagosol?

Tamagosol is a crypto-themed virtual pet game inspired by Tamagotchi. Hatch a digital creature, keep its stats healthy, stake it to earn the in-game token **$SOGO**, and evolve it into legendary forms.

## Features

- 🥚 **Hatch** — tap the egg to hatch a random creature (3 species: Solbit, Aquabit, Blazebit)
- 📊 **4 Stats** — Hunger, Energy, Mood & Hash Power decay in real time (even when closed)
- 💎 **Stake** — put your pet to work mining $SOGO passively
- ✨ **Evolve** — care well → Diamond & Whale forms; neglect it → 💀
- 💾 **Persisted** — all state saved in localStorage (progress survives reloads)
- 📱 **Mobile-first** — built for portrait phone screens

## Tech

Single `.html` file — vanilla JS, inline CSS, no external dependencies (except Google Fonts CDN).
